# Nerior Subs

Subscription and recurring payment manager for the Nerior ecosystem.

- `subs.nerior.ru` frontend
- `/api/v1` backend
- shared auth via `auth.nerior.ru`
- Planner calendar sync to calendar `Подписки`
- fixed and variable recurring payments
- groups, forecasts, AI recommendations, dashboard metrics

Local planning docs live in `/docs` and are intentionally ignored by Git.
